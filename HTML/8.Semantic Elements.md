What are Semantic Elements?

Semantic elements are HTML tags that describe the meaning of the content, not just how it looks.

They make the webpage:

Easier to read for developers

Better understood by search engines (SEO)

More accessible for screen readers

Common Semantic Elements
1. header

Represents the top section of a page or section

Usually contains logo, title, or navigation

2. footer

Represents the bottom section of a page

Contains copyright, links, contact info, etc.

3. nav

Used for navigation links/menu

Contains main links of the website

4. article

Represents independent content

Can stand alone (blog post, news article, forum post)

5. section

Groups related content into a thematic block

Used to divide the page into meaningful sections

6. aside

Represents side content related to the main content

Examples: sidebar, tips, ads, related links
